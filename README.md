# 📱 Ethio Telecom USSD Simulator (*999#)

## 📌 Description

This project is a simple **shell script application** that simulates the Ethio telecom *999# USSD menu system.

It allows users to navigate through different telecom services such as:

* Voice packages
* SMS packages
* Internet packages

The program is built using **Bash (shell scripting)** and runs in a Linux terminal.

---

## ⚙️ Features

* Menu-driven interface
* Multiple levels (Main menu → For Me/Gift → Packages → Confirmation)
* User input handling
* Looping system (keeps running until exit)
* Simple and interactive experience

---

## 🛠️ Technologies Used

* Bash (Shell Scripting)
* Linux Terminal (Ubuntu / Webminal)

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone YOUR_REPO_LINK
```

2. Go into the project folder:

```bash
cd telecom-ussd-simulator
```

3. Give permission to run the script:

```bash
chmod +x telecom.sh
```

4. Run the program:

```bash
./telecom.sh
```
